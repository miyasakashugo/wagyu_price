# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれなんでしょうか？
リモートリポジトリはgithuvなどの誰でも見れるようなリポジトリで、ローカルリポジトリはデスクトップなどの自分自身のリポジトリです。

## プッシュとマージの違いは何でしょうか？
プッシュはローカルの変更をリモートに送ることで、マージはブランチをメインに合体させるものです。

## コミットとマージの違いは何でしょうか？
コミットは変更を保存するもので、マージはブランチをマスターブランチに合体させるものです。

## コッミトメッセージはどのように書いてあげるのが最適でしょうか？
わかりやすいように端的に。

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
ローカルでマージした場合すぐにメインと合体してしまうが、プルリクエストでマージした場合レビューを受けることができる。

## コンフリクトを起こしてしまった場合、どう対処するべきですか？
開発者同士で話し合って確認する。
